利用規約

    Korioni（氷鬼）は、Minecraft上で遊べる鬼ごっこ系ミニゲーム『氷鬼』を実装できるサーバープラグインです。プレイヤーは『鬼』と『逃走者』に分かれ、鬼は逃走者を凍結させ、逃走者は協力して鬼から逃げつつ、凍った仲間を解放します。

    主な機能

    • ゲーム開始・終了コマンド (/korioni start /korioni stop)
    • ロビーやスタート地点の設定 (/korioni setlobby /korioni setstart)
    • ゲーム時間やアイテム効果時間の変更 (/korioni setconfig)
    • 鬼・逃走者の自動割り当て
    • 逃走者による凍結解除やアイテム妨害機能
    • ゲーム中スコアボード表示（残り時間・凍結人数など）
    • GUIによる設定変更（アイテム効果秒数など）

    アイテム一覧

    • 氷スティック: 触れたプレイヤーを一定時間凍結させる
    • 加速ポーション: 一定時間走る速度が上昇
    • 凍結解除アイテム: 凍った仲間を即座に解放
    • 妨害ブロック: 鬼の移動を妨げる

    攻略・戦略

    • 逃走者は凍結されない距離を保ちつつ仲間を助ける
    • 鬼は氷スティックを使い、逃走者を分断する
    • 妨害ブロックで逃走ルートを限定する
    • 加速ポーションは仲間の救助タイミングで使用

    コマンド一覧

    • /korioni start - ゲーム開始
    • /korioni stop - ゲーム終了
    • /korioni setlobby - ロビー位置設定
    • /korioni setstart - スタート地点設定
    • /korioni setconfig - ゲーム設定変更
    • /korioni reset - ゲームリセット

    スコアボード

    ゲーム中は残り時間、凍結人数、アイテム残数などが表示され、戦略の参考にできます。
