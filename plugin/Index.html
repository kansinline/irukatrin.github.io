利用規約

「氷鬼」はMinecraftサーバー専用の協力型ミニゲームプラグインです。プレイヤーは鬼と逃げに分かれて遊びます。

  • 鬼は逃げをタッチで凍結可能
  • 逃げは仲間を助けられる
  • 制限時間内に全員凍結で鬼勝利、時間切れで逃げ勝利

Minecraftサーバーに新しいミニゲームを追加し、プレイヤー同士の協力や戦略を楽しんでもらうことを目的としています。

ゲーム開始前:

  • サーバーに全員ログイン
  • 管理者が /startgame で開始
  • 最低2人以上必要

役割:

  • 鬼：逃げを凍結、鉄剣装備、スピード効果あり
  • 逃げ：仲間を助ける、スティック装備

ゲームの流れ:

  1. 鬼が逃げを追う
  2. 逃げは協力して凍結回避
  3. 凍結されたら仲間に助けてもらう
  4. 制限時間内全員凍結で鬼勝利、時間切れで逃げ勝利

  • /startgame : ゲーム開始
  • /resetgame : ゲームリセット
  • /settime <秒> : 制限時間設定
  • /rules : ルール表示
